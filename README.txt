FeelsBook is an application that allows the user to log their feelings throughout the day with optional comments.

The class Feel has String mood, Date date and String comment attributes that are associated with each entry. The comment can range inclusively between 0 and 100 characters. The date is display in iso1806 format. The mood is the user's choice from the options: love, joy, surprise, anger, sadness, fear.
